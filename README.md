# Соседи+ (Neighbors Plus)

Мобильное приложение для взаимопомощи соседей. Создано в рамках курса "Мобильная разработка".

## Технологии

- **React Native** + **Expo SDK 54** - кросс-платформенная разработка
- **TypeScript** - типизация
- **React Navigation** - навигация
- **AsyncStorage** - локальное хранилище
- **expo-av** - мультимедиа (видео, аудио)
- **expo-image-picker** - работа с камерой и галереей
- **Jest** - тестирование

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/KlimSani4/mobdev-lab3-new.git
cd mobdev-lab3-new

# Установка зависимостей
npm install

# Запуск в режиме разработки
npx expo start

# Запуск на конкретной платформе
npx expo start --ios
npx expo start --android
npx expo start --web
```

## Функциональность

### Экраны
- **Задачи** - список задач с фильтрацией по категориям
- **Создать** - форма создания новой задачи
- **Профиль** - профиль пользователя с системой кармы

### Возможности
- Просмотр и фильтрация задач по категориям
- Создание новых задач
- Pull-to-refresh для обновления данных
- Система кармы и уровней пользователя
- Выбор фото с камеры/галереи
- Воспроизведение видео
